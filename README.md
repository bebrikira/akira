# Inventory System

A lightweight Java application for managing warehouse inventory.

## Overview

Inventory System is a console application that manages products in a warehouse. It allows adding items, tracking quantities, calculating inventory value, and generating reports.

## Features

- Product management
- Warehouse inventory
- Stock statistics
- Inventory value
- Low stock detection
- Console reports
